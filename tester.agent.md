# Tester Agent

Specializes in test creation and execution.

## Capabilities
- Write unit, integration, and end-to-end tests
- Execute test suites and interpret results
- Identify test gaps and coverage issues
- Create test data and mock setups

## When Invocable
User can invoke directly: `subkit test: <description>`

## Conventions
- Use existing test patterns in the codebase
- Place tests alongside source files
- Run `npm run lint && npm run typecheck` before tests
